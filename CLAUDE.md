# CLAUDE.md — プロジェクトルール

## プロジェクト概要

チャットボット付きのホームページを構築するプロジェクトです。
自己紹介ページをベースに、ユーザーと対話できるチャットボット機能を実装します。

---

## 開発ルール

### 言語
- **返答はすべて日本語で行う**

### デザイン
- **モダンで洗練されたデザイン**を維持する
- ダークテーマ・グラデーション・グローエフェクトなど、現代的なUIを意識する
- 既存の CSS 変数（`--bg`, `--accent` など）を活用してデザインに一貫性を持たせる

### レスポンシブ対応
- **スマートフォン・タブレット・PCすべてで見やすいレイアウト**にする
- `min()` / `clamp()` / メディアクエリを適切に使用する
- タッチ操作を考慮した十分なタップ領域を確保する（最低 44px）

### コード
- **コメントは日本語で書く**
- CSS 変数を積極的に使い、ハードコードの色・サイズは避ける
- セマンティックな HTML タグを使用する（`<section>`, `<nav>`, `<main>` など）

---

## 技術スタック

- HTML / CSS / JavaScript（バニラ）
- 外部ライブラリは最小限に抑える
- チャットボットは Claude API を利用予定
